# Agent Behavioral Rules

- After making changes to the code, DO NOT push to GitHub automatically. Wait for the user to review the changes before pushing.
- 要了解用户背景或项目历史信息，去读 `~/llm-wiki/`（个人知识库：`wiki/` 结构化知识 + `raw/` 对话记录归档 + `dev/wiki/` 开发项目知识层）。新会话可先读 `~/llm-wiki/wiki/index.md` 或 `~/llm-wiki/dev/wiki/index.md` 恢复上下文。
